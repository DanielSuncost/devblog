# DevBlog skill registry entry for Codex/agentic coding tools

Use DevBlog for background-tracked development logs, not generic blogging.

Shared command contract:
- `devblog init --repo .`
- `devblog track --repo . --once`
- `devblog entry --repo .`
- `devblog status --repo .`

Shared files:
- `.devblog/config.json`
- `.devblog/state.json`
- `.devblog/ledger.jsonl`
- `.devblog/entries/*.md`

Keep compatibility with Hermes, Claude Code, Codex, and Charon.
